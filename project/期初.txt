方案二 拉麵王 (名稱待議

理由 : 
因為跟一群朋友很喜歡吃鱔魚意麵
然後每次到台南一帶旅遊時都會找找還有哪家鱔魚意麵沒吃過
所以想要做一個網頁可以記錄吃過了哪間但由於鱔魚意麵受眾真的太小
同時間也想到"拉麵"(日式)其實也有一樣的需求
並且受眾更多因此覺得可以試著做做看拉麵以及
評價時加入一點趣味要素例如評價了3間拉麵會升一個等級並且讓所有使用者看到比較
然後也是有這個要素在才會有想取成拉麵王的想法
並且製作後台讓管理員(們) 可以自由新增店家進資料庫(會員可以申請變成管理員)
標語 :
1.想要找到最好吃的拉麵嗎？那就去找吧！

資料庫:

member 
 _id int 索引
 permission int 權限 0會員 1管理員 99超級管理員
 gener int 性別 1男2女
 name string 姓名
 account string 帳號
 password string 密碼(hashSync)
 isdelete bool  是否刪除 0未刪除 1已刪除
